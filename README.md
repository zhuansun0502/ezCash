# ezCash

This is an iOS application written in Swift and supported by Firebase that aims to reduce the efforts when trading used items.







The home screen is a map view, enables the users to see what available trading options around them base on their current location.

<img src="https://github.com/zhuansun0502/ezCash/blob/master/Demo/WeChat%20Image_20210113101131.jpg" height="700px" width="350px">

Before being able to upload any images, create an account first using your email followed by setting up your password.
